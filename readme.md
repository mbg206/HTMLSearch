# HTMLSearch

A basic search engine/tool for searching through HTML files.

---

## Usage

Set path in `path.json`

Open `index.js`

Open `index.html` in your browser

---

### Port

If you need to change the port, change line 35 of `index.js` AND line 4 in `index.html`.
